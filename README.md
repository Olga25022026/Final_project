Название проекта: Анализ закупок запасных частей для ремонта локомотивов. 

У нас есть "сырые данные" по закупке запасных частей за 4 квартал 2025 г. Необходимо обработать данные для последующей загрузки и демонстрации на дашборде в Datalens. 

Основная цель дашборда: мониторинг и анализ эффективности закупочной деятельности, повышение прозрачности закупок и снижение рисков срыва ремонтной программы локомотивов

Пройдемся по основным чартам: 
1.	Общая цифра закупки отражает оценку исполнения бюджета.
2.	Кол-во ТМЦ - объем закупаемой номенклатуры.
3.	ТОП поставщики, график визуализирует структуру закупок в разрезе поставщиков. Цель анализа - выявление критической зависимости от конкретных контрагентов. Задача - оценить концентрацию закупок для диверсификации поставщиков и снижения рисков монополии.
4.	Графики по филиалам отражают географию закупок, служат основой для планирования финансирования филиалов, оптимизации складских запасов, а также для планирования графиков инвентаризации и контроля за товарными остатками на местах.
5.	Источник закупки в % отражает уровень централизации закупочной функции. Анализ соотношения закупок через головной офис и силами филиалов необходим для принятия управленческих решений о перераспределении полномочий, оптимизации логистики.
6.	Кол-во ТМЦ в зависимости от статуса показывает узкие места в работе с поставщиками, проблематику в случае отказа от поставки, это сигнал для отдела снабжения о необходимости поиска новых поставщиков, изменений условий закупочных процедур.
7.	Сумма закупки по месяцам отражает сезонное планирование и равномерность распределения нагрузки на филиалы, позволяет синхронизировать планы закупок с планами ремонта локомотивов, управлять товарными запасами для предотвращения затоваривания складов.
